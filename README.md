# dextra

Intructions to run the project:

# Backend
the backend will run on the address localhost:5000
- cd backend;
- npm install;
- node_modules/.bin/babel-node app.js;

# Unittests
- cd backend;
- npm install;
- npm test

# Frontend
the frontend will run at the address localhost:4200
- cd frontend/menuFront;
- npm install
- ng serve
